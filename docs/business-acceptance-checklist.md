# 业务验收样本清单

用于上线前或每次关键 PR 合并后的人工回归。以下样本不使用真实机构名称，执行时可替换为已脱敏的业务客户。

## 样本 1：正常授信

- 机构名称：样本机构 A 正常合作诊所
- 合作阶段：合作 ≥ 6 个月
- 回款订单：已有，8 单
- 近 6 个月采购：60000 / 58000 / 61000 / 59000 / 62000 / 60000
- 最长断档：20 天
- 履约：无当前逾期，历史超账期为无
- 资质：完整
- 公共风险核验：未发现明显风险，人工确认公共信用正常
- 预期结果：正常授信；等级 A 或 A+；无需特批；结果页显示最终评估与已人工确认。

## 样本 2：额度或账期超规则特批

- 机构名称：样本机构 B 高额度申请诊所
- 合作阶段：合作 ≥ 6 个月
- 回款订单：已有，10 单
- 近 6 个月采购：40000 / 42000 / 41000 / 39000 / 43000 / 41000
- 申请账期：45 天
- 申请额度：80000
- 履约：无当前逾期，历史超账期为无
- 公共风险核验：人工确认正常
- 预期结果：正常授信，需特批；特批原因包含申请账期超过等级上限或申请额度超过稳定月均销量。

## 样本 3：高风险但非红线

- 机构名称：样本机构 C 行政处罚线索诊所
- 合作阶段：合作 3-6 个月
- 回款订单：已有，4 单
- 近 6 个月采购：30000 / 0 / 28000 / 32000 / 0 / 31000
- 最长断档：65 天
- 履约：历史超账期 4-7 天
- 公共风险核验：发现行政处罚或重大医美处罚线索，人工确认公共信用为中等风险
- 预期结果：谨慎短账期或谨慎短账期，需特批；最高等级不超过 C 或 D；结果页展示核验状态、证据摘要、复核人和时间戳。

## 样本 4：失信 / 严重违法红线

- 机构名称：样本机构 D 失信命中机构
- 合作阶段：合作 ≥ 6 个月
- 回款订单：已有，6 单
- 近 6 个月采购：50000 / 52000 / 51000 / 49000 / 50000 / 52000
- 公共风险核验：发现失信被执行人或严重违法失信线索
- 人工确认：采用系统建议或人工改判为“失信 / 严重违法”
- 预期结果：等级 E；不建议授信；账期 0 天；额度 0；历史记录同步显示更新后的 E 级结果。

## 样本 5：核验失败后补跑

- 机构名称：样本机构 E 核验补跑诊所
- 操作前提：模拟网络异常、Function 暂时失败或后台核验返回 failed
- 操作路径：保存评估记录后进入核验页，点击“重新核验”
- 预期结果：生成新的 pending / running 日志；完成后结果页显示最新核验状态；人工确认后形成最终评估。

## 每条样本的通用检查

- 基础、采购、履约、核验、结果五个步骤可顺畅切换。
- 修改采购金额后，稳定月均销量和建议额度实时变化。
- 保存记录后，后台核验状态可见。
- 证据卡片可打开原文链接。
- 保存人工确认日志后，风控字段和历史记录同步更新。
- 证据附件支持 PNG / PDF 上传，签名链接可打开。
- 结果页可复制审批摘要，并可导出审批长截图。
- 手机宽度下无横向滚动，控制台无应用报错。
